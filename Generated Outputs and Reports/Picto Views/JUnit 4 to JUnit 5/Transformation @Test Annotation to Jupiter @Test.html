Transformation: @Test Annotation to Jupiter @Test

Migration relation type: Transformation
Migration scope: JUnit 4 => JUnit 5 / JUnit Jupiter and JUnit Platform

Applicability conditions:

  • ABSENT on @Test(expected), @Test(timeout): Applies only to simple @Test usage without expected or timeout attributes; otherwise use exception o r timeout transformations.

Transformation intent

Source feature: @Test annotation
Target feature(s): org.junit.jupiter.api.Test
Automation: OperatorHelp / MIX
Relation: Both annotations declare a test method, but the package changes from org.junit.Test to org.junit.jup iter.api.Test and JUnit 4 attributes expected and timeout are not part of Jupiter @Test.

Operators: Migration engineer / test developer

Realization strategy


1. [A] Replace import org.junit.Test with import org.junit.jupiter.api.Test.



2. [A] If @Test has no expected or timeout attributes, leave the annotation shape unchanged.



3. [A] If expected is present, delegate to the @Test(expected) to assertThrows correspondence.



4. [A] If timeout is present, delegate to the @Test(timeout) correspondence.



5. [M] Verify that method visibility and runner/rule semantics do not depend on JUnit 4 behavior.
