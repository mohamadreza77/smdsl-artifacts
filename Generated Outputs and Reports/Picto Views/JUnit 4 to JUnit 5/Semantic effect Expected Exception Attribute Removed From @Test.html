Semantic effect

Affected target feature(s): assertThrows, assertThrowsExactly, assertThrows block
Rationale: The expected attribute is not part of Jupiter @Test. Migrated code must introduce assertThrows aroun d the intended throwing code and explicitly check message or cause when required.
