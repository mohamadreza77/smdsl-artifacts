Transformation: type-level ConstructorBinding to constructor-level or removal

Migration relation type: Transformation
Migration scope: Spring Boot 2.7.x / late Spring Boot 2 application => Spring Boot 3.0 application / migration target

Applicability conditions: None specified.

Transformation intent

Source feature: Type-level @ConstructorBinding
Target feature(s): Constructor-level @ConstructorBinding only when needed
Automation: FullAutomation / EASY
Relation: @ConstructorBinding is no longer needed at the type level; it should be removed, except that it may still be used on a selected constructor when multiple constructors exist.

Realization strategy


1. [A] find type-level @ConstructorBinding on @ConfigurationProperties classes



2. [A] remove it when the type has one constructor or is a record



3. [NONE] if multiple constructors exist

    3.1. [A] move @ConstructorBinding to the intended constructor





4. [A] run configuration binding tests
