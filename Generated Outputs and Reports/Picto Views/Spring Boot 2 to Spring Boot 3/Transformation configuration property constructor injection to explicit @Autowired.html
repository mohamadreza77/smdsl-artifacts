Transformation: configuration property constructor injection to explicit @Autowired

Migration relation type: Transformation
Migration scope: Spring Boot 2.7.x / late Spring Boot 2 application => Spring Boot 3.0 application / migration target

Applicability conditions: None specified.

Transformation intent

Source feature: Constructor dependency injection in @ConfigurationProperties without @Autowired
Target feature(s): @Autowired for dependency-injected configuration property constructors
Automation: OperatorHelp / MIX
Relation: If a @ConfigurationProperties constructor is meant for dependency injection rather than property bin ding, it must be annotated with @Autowired to avoid being treated as a binding constructor.

Operators: Application Developer 2

Realization strategy


1. [A] find @ConfigurationProperties classes with constructor parameters that look like beans



2. [M] decide whether each constructor is binding or dependency injection



3. [A] add @Autowired for dependency-injection constructors



4. [A] run application context startup tests
