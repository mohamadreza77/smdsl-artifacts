Semantic effect

Effect kind: DEFAULT_VALUE_CHANGED
Affected source feature(s): Implicit trailing slash URL matching
Affected target feature(s): Explicit URL mappings or redirect/rewrite strategy for trailing slash, Temporary global trailing-slash compatibility configuration
Rationale: Spring Framework 6 disables trailing slash matching by default; routes that previously matched /path / may now return 404 unless handled explicitly.
