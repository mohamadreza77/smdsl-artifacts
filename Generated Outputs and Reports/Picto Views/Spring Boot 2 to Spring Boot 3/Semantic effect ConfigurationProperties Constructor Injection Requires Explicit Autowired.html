Semantic effect: ConfigurationProperties Constructor Injection Requires Explicit Autowired

Migration relation type: SemanticEffect
Migration scope: Spring Boot 2.7.x / late Spring Boot 2 application => Spring Boot 3.0 application / migration target

Applicability conditions: None specified.

Semantic effect

Effect kind: VALIDATION_REQUIRED
Affected source feature(s): Constructor dependency injection in @ConfigurationProperties without @Autowired
Affected target feature(s): @Autowired for dependency-injected configuration property constructors
Rationale: A constructor in a @ConfigurationProperties class may be treated as a binding constructor unless dep endency injection is marked explicitly with @Autowired.
